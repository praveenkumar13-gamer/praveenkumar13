## 🚀 Featured Projects

### 🪙 Vision-Based Fake Indian Currency Detection with Real-Time Voice Feedback
> *Research Paper · ICACCT'26*

A real-time computer vision system that detects counterfeit Indian currency notes and delivers instant audio feedback — making the tool accessible for visually impaired users.

- 📌 **Tech:** Python, OpenCV, CNN, Text-to-Speech (pyttsx3 / gTTS)
- 🎯 **Impact:** Combines vision AI with accessibility — identifies fake notes in real time
- 📰 **Published:** International Conference on Advanced Computing & Communication Trends 2026

### 🤟 Sign Language Recognition System
> *Real-time hand gesture recognition for bridging communication gaps*

A computer vision system that detects and classifies American Sign Language (ASL) gestures in real time using hand landmark tracking — enabling seamless communication for the hearing impaired.

- 📌 **Tech:** Python, OpenCV, MediaPipe, TensorFlow/Keras, CNN
- 🎯 **Impact:** Classifies 26 ASL alphabets with live webcam feed and on-screen text output
